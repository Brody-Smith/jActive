jActive

jActive is a simple, yet flexible jquery plugin for applying a class based on the specified criteria.

View on GitHub Download .zip Download .tar.gz

How To Use

Simply included the jActive plugin after jQuery.

<!--  jQuery 1.7+  -->
<script src="jquery-1.11.1.min.js"></script>

<!--  jActive Plugin  -->
<script src="src/jactive.min.js"></script>

Call the plugin.

<script type="text/javascript">
    $(function(){
        $('[data-jactive]').jActive();
    });
</script>

The next step is to specify data-* attributes on anything you want to apply a class to when the criteria is met..

jActive relys on these attributes to determine when the active class should be added. At the time of writing there are three different values you can supply. As an example let's imagine we want to set an active class for a settings page:

String
Strictly matches '/settings' 
data-jactive="/settings"
String (with alternatives)
Matches either '/settings' or '/account-settings'. 
data-jactive="/settings || /account-settings" OR data-jactive="/settings||/account-settings"
Regular Expression
Matches '/settings' and any other url that starts with settings ie. '/settings/account' or '/settings/profile' 
data-jactive="/^\/settings(\/[\s\S]+)?$/"
