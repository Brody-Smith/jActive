jActive

A simple, yet flexible jquery plugin for applying a class based on the specified criteria.

What is jActive?

jActive is an agnostic jQuery plugin, in that it wasnâ€™t built for any particular scenario, and can be used in many different scenarios. By default jActive is set to apply an active class based on the URL, however by no means is this the only possibility!

For examples on different ways you can utilise jActive in your projects please refer to the demo page.

Features

  • Kick-ass Name
  • Multiple String Checks
  • Regular Expression Checks
  • Compatible with all modern browsers

Get Started

HTML

You will need specify data-jactive attributes on anything you want to apply a class to when the criteria is met.

As an example let's imagine we want to set an active class for a /settings page, and account for alternatives such as /settings/account or /settings-account.

There are currently three different values you can supply in the attributes.

  • String: Strictly matches against the string
  • String (with alternatives): Matches either options seperated with ||
  • Regular Expression: Matches the supplied regex

    <!-- String -->
    <a data-jactive="/settings">Settings</a>

    <!-- String (with alternatives) -->
    <a data-jactive="/settings || /settings-account">Settings</a>

    <!-- Regular Expression -->
    <a data-jactive="/^\/settings(\/[\s\S]+)?$/">Settings</a>

Include the jActive plugin after jQuery (1.7+). Ideally before the closing </body> tag.


    <!--  jQuery 1.7+  -->
    <script type="text/javascript" src="jquery-1.11.1.min.js"></script>

    <!--  jActive Plugin  -->
    <script type="text/javascript" src="src/jactive.min.js"></script>

JavaScript

After including the jActive plugin call it and specify any options you want to change. jActive will work without any changes to the options specified, they are displayed as a reference to the default values.

  • Class: The class that gets applied
  • Name: The name of the data attribute you want to use
  • Match: What the criteria will be checked against

<script type="text/javascript">
    $(function(){
        $('[data-jactive]').jActive({
            class : 'active',
            name  : 'jactive',
            match :  window.location.pathname
        });
    });
</script>
